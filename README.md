# Raylib_Tetris

A C++ + CMake Raylib minigame.

## To Play
In the Root folder:
```sh
./build/Debug/Source/Main/My_00
```

## Controls:
Left, Down and Right arrows.
If Game over, press any key :D

## Quit
Esc or click in the Window X
